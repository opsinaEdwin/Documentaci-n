Responsive

El diseño web adaptativo consiste en crear páginas web que se vean bien en todos los dispositivos. Un diseño web responsivo se ajusta automáticamente a los distintos tamaños de pantalla y ventanas gráficas.

El diseño web adaptable consiste en utilizar HTML y CSS para redimensionar, ocultar, reducir o ampliar automáticamente un sitio web para que se vea bien en todos los dispositivos (ordenadores de sobremesa, tabletas y teléfonos)

Sesion de usuario

Las sesiones permiten almacenar datos del usuario en el servidor de forma temporal y segura. Por ejemplo, datos como su nombre, ID de usuario, o incluso preferencias verificar la identidad del usuario en cada página, ayudando a garantizar que solo los usuarios autorizados accedan a sus respectivas cuentas y áreas restringidas

Para esto se deben generar formularios que envien la informacion del usuario a la base de datos, esto permite identificar que usuarios estan usando la aplicación y personalizar la aplicacion para cada persona
